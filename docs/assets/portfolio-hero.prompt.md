# Portfolio hero prompt

`README.md`의 hero는 GPT Image built-in mode로 생성한 포트폴리오 콘셉트 비주얼이다. 실제 애플리케이션 스크린샷이나 worker 출력이 아니다. 3D로 드레이프된 원단은 생성된 flat tileable texture가 물리 제품에 적용되는 모습을 상상한 포트폴리오 표현이며, 벌 motif도 샘플 시각 요소이지 브랜드 로고가 아니다.

프롬프트는 `../seamless-tile`의 대표 시나리오인 대각 repp stripe, lane을 따르는 motif, 벡터 반복에서 직조 원단으로 이어지는 finalize 개념을 참고했다.

```text
Use case: stylized-concept
Asset type: wide portfolio README hero image
Primary request: visualize the core seamless textile workflow as one elegant editorial composition: a precise flat vector repeat gradually becoming richly woven bespoke necktie fabric.
Scene/backdrop: minimal warm off-white studio surface, clean and architectural.
Subject: one continuous textile ribbon crosses the wide frame; on one side it reads as a crisp, mathematically aligned square repeat with visible but subtle modular tile logic, and it transitions naturally into folded, tactile silk-jacquard fabric on the other side. Use a classic diagonal repp stripe necktie pattern with a deep navy ground, narrow silver and antique-gold stripes, plus small refined single-color bee motifs arranged along selected diagonal lanes. The repeat must look continuous at every implied boundary, with no broken stripes or clipped motifs.
Style/medium: premium editorial product photography blended seamlessly with restrained vector-design visualization; sophisticated Korean fashion-tech portfolio aesthetic, not sci-fi.
Composition/framing: 16:9 cinematic wide banner, balanced center composition, generous quiet margins, no devices, no people.
Lighting/mood: soft directional studio light, calm, precise, premium.
Color palette: deep navy, warm off-white, graphite black, muted silver, restrained antique gold.
Materials/textures: realistic fine silk weave and subtle jacquard relief; crisp vector geometry before the transition.
Constraints: communicate prompt-to-pattern-to-fabric without showing words; no text, no letters, no numbers, no logos, no trademarks, no watermark; no UI screens; no random floral elements; no visible seams, broken repeats, or malformed motifs.
Avoid: cyberpunk glow, generic AI imagery, floating code, circuit boards, excessive props, glossy plastic, clutter.
```
