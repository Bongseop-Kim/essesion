# 국제화·현지화

한국어 우선 표기 규칙. 문구 톤은 [voice-and-tone.md](./voice-and-tone.md), 숫자·단위 문구는 [writing.md](./writing.md).

## 한국어 줄바꿈

- `word-break: keep-all` — `html`에 적용 완료(어절 단위 줄바꿈, 단어 중간 끊김 방지).
- 금칙 처리(줄 끝·시작에 오면 안 되는 문자) 존중 — 문장부호가 홀로 넘어가지 않도록.
- 번역 확장(같은 의미의 다른 언어가 더 길어짐)에 대비해 **고정폭 컨테이너를 지양**한다.

## 통화

- 원화 `₩` 기호 + 천단위 콤마. `Intl.NumberFormat("ko-KR")` 사용.
- 예: `₩1,200,000`. 임의 문자열 조립·수동 콤마 삽입 금지.

## 날짜·시간

- 날짜: `YYYY.MM.DD`(예: `2026.07.08`).
- 상대시간: `n분 전`·`n시간 전`·`어제` 기준(최근 항목). 오래된 항목은 절대 날짜.

## 숫자 (admin 테이블)

- 숫자 열은 `font-variant-numeric: tabular-nums` + **우측 정렬**(자릿수 정렬로 스캔 용이).
- 텍스트 열은 좌측 정렬 유지. → [layout.md](./layout.md)
